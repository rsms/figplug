figplug

figplug is a small program for building Figma plugins. It offers all the things you need for most projects: TypeScript, React/JSX, asset bundling, plugin manifest generation, etc.

Use

Create or initialize a new plugin project: figplug init

Create a new plugin with UI: figplug init -ui

Build a plugin, rebuilding it automatically as your source files change: figplug build -w

Build a plugin with optimizations: figplug build -O

See help for more options: figplug help init
figplug help build

Install

npm install -g https://rsms.me/figplug/fp-0.0.3.tgz

Note: Since this is pre-release beta software, figplug is not yet available from npm.

To uninstall, run npm uninstall -g figplug
